## 🔬 技術原理

### 量子電路構建

我們為每個文本構建專門的量子電路：

```python
# 1. 初始化疊加態 - 模擬敘事多重可能性
for i in range(n_qubits):
    qc.h(i)

# 2. 語義權重編碼 - 基於TF-IDF權重
for i, weight in enumerate(word_weights):
    theta = weight * np.pi + np.pi/4
    phi = len(word) / 10 * np.pi
    qc.ry(theta, i)
    qc.rz(phi, i)

# 3. 語義糾纏創建 - 模擬詞彙間關聯
for i in range(n_qubits - 1):
    for j in range(i + 1, n_qubits):
        similarity = calculate_word_similarity(words[i], words[j])
        if similarity > 0.3:
            angle = similarity * np.pi / 2
            qc.cry(angle, i, j)
```
