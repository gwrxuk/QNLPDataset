### **动态量子位元分配演算法详解**

#### **演算法流程**

def calculate_qubits(pos_tags):
    unique_pos = set(pos_tags)
    num_qubits = min(4, max(2, len(unique_pos)))
    return num_qubits

### **量子门參數計算公式**

#### **旋转角度計算**
```python
angle = base_angle × frequency_ratio × scaling_factor

其中:
- base_angle = category_map[pos_type]  # 詞性基础角度
- frequency_ratio = count / total_words  # 詞性频率比例
- scaling_factor = π/4  # 缩放因子
```

### **量子電路构建過程**

#### **步骤1: 初始化疊加态**
```python
for i in range(num_qubits):
    circuit.h(i)  # 所有量子位元进入疊加态
```

#### **步骤2: 詞性特征編碼**
```python
pos_counts = Counter(pos_tags)
for i, (pos, count) in enumerate(list(pos_counts.items())[:num_qubits]):
    if pos in category_map:
        angle = category_map[pos] * (count/len(pos_tags)) * π/4
        circuit.ry(angle, i)  # 旋转门編碼詞性資訊
```

#### **步骤3: 创建量子糾纏**
```python
if num_qubits > 1:
    circuit.cx(0, 1)  # 在前两個量子位元间创建糾纏
```
